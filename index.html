Command Line Interface (CLI)

    Git Verions Status:

  1. git --version

    2. List files and directories (ls):

  2. ls

    3. Change directory (cd):

  3. cd
  4. cd Documents
  5. cd ~
  6. cd Documents/Learning/javaScript-File
  7. cd..

    8. Up to (Back), Down to (front):

    10. Right to (Edit), Left to (Edit):

  12. Alt + mouse to select specific location of command
  13. Ctrl + A to select the beginning location of command
  14. Ctrl + E to select the end location of command
  15. Ctrl + U to clear the command

    16. Create New Folder:

  16. mkdir Folder-Name
  17. cd Folder-Name
  18. mkdir "1.5: Folder Name"
  19. cd 1.5\ Folder \ Name

    20. Create New File:

  20. touch Text.txt

    21. Open File:

  21. start Text.txt

    22. Open File Using VS Code:

  22. code Text.txt

    23. Remove File:

  23. rm Text.txt

    24. Print working directory (pwd) to Show up the current folder location:

  24. pwd

    25. Delete All Files:

  25. rm *

    26. Delete Directory Folder:

  26. rm -r Folder-Name/
  27. rm -r "1.5: Folder Name"/

    28. Clear Display:

  28. clear

    29. Eixt Terminal or Bash:

  29. exit
